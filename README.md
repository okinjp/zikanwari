# SFCの時間割と現在の授業の終了までの時間を出してくれるシェルスクリプト(for Bash)

### 使い方
- terminalを開く
- `cd`
- `git clone https://github.com/okinjp/zikanwari_bash.git`
- `cd zikanwari_bash`
- `sudo cp ./zikan /usr/local/bin/`
- `sudo chmod +x /usr/local/bin/zikan`
- `zikan`
