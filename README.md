# SFCの時間割と現在の授業の終了までの時間を出してくれるシェルスクリプト

### 使い方
- terminalを開く
- `cd`
- `git clone https://github.com/nekosfc/zikanwari.git`
- `cd zikanwari`
- `sudo cp ./zikan /usr/local/bin/`
- `sudo chmod +x /usr/local/bin/zikan`
- `zikan`

### その他
- mac用です
- シェルスクリプト書くの初めてなのでガバガバです(言い訳)
